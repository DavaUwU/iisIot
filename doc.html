Databáze



Instalace

  • nainstalovat LAMP stack (linux server, apache, mysql, php), symfony a composer

  • extensions pro php:

    • php-gd php-json php-mbstring php-intl php-mysql php-xml php-zip

  • php >= 8.1, symfony 6

  • umístit projekt do /var/www/html/ (např.: git clone <url_repozitare>)

  • nastavit virtualhost

  • v .env nastavit sql server

  • přidat do databáze uživatele app

  • v kořenové složce projektu spustit následující příkazy:

    • composer install

    • symfony console doctrine:database:create

    • symfony console make:migration

    • symfony console doctrine:migrations:migrate

    • symfony console doctrine:fixtures:load
